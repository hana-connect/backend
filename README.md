## 🚀 프로젝트 소개

**하나커넥트**는 금융을 통해 가족을 연결하고, 손주에게 마음과 미래를 함께 전하는 참여형 금융 서비스입니다.

액티브 시니어의 금융 자산은 4300조로 추정되며, 주요 관심사 중 하나는 손주입니다.  
최근 5년간 미성년자에게 증여된 자산 규모는 약 3조 8000억원에 달합니다.

하지만 이러한 자산 전달은 단순한 숫자로만 남는 경우가 많습니다.

**하나커넥트는 돈 + 감정 + 기록을 함께 전달하는 금융 서비스**로,  
기존의 일방향 금융 경험을 넘어 가족 간 관계를 연결합니다.

## ✨ 핵심 기능

### 1️⃣ 청약 납입 및 리포트
- 조부모 → 손주 청약 납입
- 납입 금액 초과 시 안내
- 선납 및 자동 납입 기능
- 납입 히스토리 기반 리포트 제공
- UI 레벨에 따라 성장하는 집 시각화

### 2️⃣ 적금 메시지 & 우체통
- 송금 시 메시지 작성
- 최근 메시지 3건 확인
- 만기 시 메시지 모음 제공 (적금 우체통)

### 3️⃣ AI 퀴즈
- 아이 활동 기반 퀴즈 생성
- 정답 시 리워드 지급

### 4️⃣ AI 자산관리
- 자산 통합 조회
- 금융 성향 기반 자산 추천
- 용돈 금액 추천 및 즉시 송금
